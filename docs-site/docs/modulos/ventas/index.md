# Módulo de Ventas

El módulo de Ventas te permite analizar el comportamiento de ventas de todas tus tiendas en tiempo real.

## Vista General

Al acceder a **Ventas** desde el menú principal, verás un resumen de todas tus ubicaciones con:

- Total de ventas por tienda
- Comparativo vs período anterior
- Indicadores de tendencia

## Características

### Dashboard por Tienda

Cada tienda tiene su propio dashboard con:

- **KPIs principales** - Ventas del día, semana y mes
- **Gráfico de tendencias** - Evolución de ventas en el tiempo
- **Top productos** - Los más vendidos en el período
- **Análisis por categoría** - Distribución de ventas

### Transacciones

Accede al detalle de cada transacción:

- Fecha y hora
- Productos vendidos
- Cantidades y montos
- Cliente (si aplica)

### Reportes

Genera reportes de ventas con filtros por:

- Rango de fechas
- Tienda
- Categoría de producto
- Producto específico

## Navegación

```
Ventas
├── Resumen (todas las tiendas)
└── Tienda específica
    ├── Dashboard
    ├── Transacciones
    └── Detalle de producto
```
