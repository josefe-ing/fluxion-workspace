# Reportes de Ventas

Genera reportes detallados de ventas para análisis y toma de decisiones.

## Tipos de Reportes

### Reporte de Ventas por Período

Muestra el total de ventas agrupado por:
- Día
- Semana
- Mes

### Reporte de Ventas por Producto

Detalle de ventas a nivel de SKU incluyendo:
- Código de producto
- Descripción
- Cantidad vendida
- Monto total
- Porcentaje de participación

### Reporte de Ventas por Categoría

Análisis de ventas agrupado por categoría de producto.

### Reporte Comparativo

Compara ventas entre:
- Períodos diferentes
- Tiendas diferentes
- Productos similares

## Cómo Generar un Reporte

1. Selecciona el **tipo de reporte**
2. Configura los **filtros** (fechas, tienda, categoría)
3. Click en **Generar**
4. Visualiza los resultados
5. **Exporta** si es necesario

## Formatos de Exportación

| Formato | Uso recomendado |
|---------|-----------------|
| **Excel** | Análisis adicional, pivot tables |
| **PDF** | Presentaciones, archivo |
