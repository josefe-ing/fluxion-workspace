# Dashboard de Ventas

El dashboard de ventas proporciona una vista completa del rendimiento de ventas de cada tienda.

## Acceso

1. Ve a **Ventas** en el menú lateral
2. Haz click en una tienda del listado
3. Serás llevado al dashboard de esa tienda

## Componentes del Dashboard

### Tarjetas de KPIs

En la parte superior verás tarjetas con métricas clave:

| Métrica | Descripción |
|---------|-------------|
| **Ventas Hoy** | Total de ventas del día actual |
| **Ventas Semana** | Acumulado de la semana |
| **Ventas Mes** | Acumulado del mes |
| **Variación** | Comparativo vs período anterior |

### Gráfico de Tendencias

Visualiza la evolución de ventas:

- **Diario** - Ventas día a día
- **Semanal** - Agregado por semana
- **Mensual** - Agregado por mes

Puedes cambiar el período usando el selector de fechas.

### Top Productos

Lista de los productos más vendidos mostrando:

- Nombre del producto
- Cantidad vendida
- Monto total
- Porcentaje del total

### Ventas por Categoría

Gráfico de pie/dona que muestra la distribución de ventas por categoría de producto.

## Filtros Disponibles

- **Rango de fechas** - Selecciona el período a analizar
- **Categoría** - Filtra por categoría de producto
- **Producto** - Busca un producto específico

## Acciones

### Ver Detalle de Producto

Haz click en cualquier producto para ver:

- Histórico de ventas
- Tendencia de demanda
- Stock actual
- Clasificación ABC/XYZ

### Exportar Datos

Usa el botón de exportar para descargar:

- Reporte en Excel
- Reporte en PDF
