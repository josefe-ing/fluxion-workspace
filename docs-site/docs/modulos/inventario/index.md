# Módulo de Inventario

El módulo de Inventario te permite monitorear los niveles de stock en tiempo real y recibir alertas proactivas.

## Vista General

Al acceder a **Inventario** verás un resumen de todas tus ubicaciones con:

- Valor total del inventario
- Productos con stock crítico
- Productos con sobre-stock
- Rotación de inventario

## Características

### Stock Actual

Visualiza el inventario actual de cada tienda:

- Cantidad disponible por producto
- Cobertura en días
- Clasificación ABC del producto
- Estado del stock (normal, bajo, crítico)

### Alertas de Inventario

El sistema genera alertas automáticas para:

- **Stock crítico** - Productos por debajo del punto de reorden
- **Quiebre de stock** - Productos sin existencia
- **Sobre-stock** - Productos con exceso de inventario
- **Próximos a vencer** - Productos cercanos a fecha de expiración

### Historial de Movimientos

Rastrea todos los movimientos de inventario:

- Entradas (compras, transferencias)
- Salidas (ventas, mermas)
- Ajustes de inventario

## Navegación

```
Inventario
├── Resumen (todas las tiendas)
└── Tienda específica
    ├── Stock actual
    ├── Alertas
    └── Historial
```
