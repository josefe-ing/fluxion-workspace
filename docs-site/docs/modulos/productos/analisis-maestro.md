# Análisis Maestro

Vista integral de todo tu catálogo de productos con métricas consolidadas.

## Descripción

El Análisis Maestro te da una visión completa de cada producto en tu catálogo, consolidando información de:

- Datos maestros del producto
- Métricas de venta históricas
- Niveles de stock actuales
- Clasificaciones ABC y XYZ

## Tabla Principal

| Columna | Descripción |
|---------|-------------|
| **Código** | SKU único del producto |
| **Producto** | Nombre/descripción |
| **Categoría** | Categoría del producto |
| **Venta 30d** | Ventas últimos 30 días |
| **Stock Total** | Stock consolidado todas las tiendas |
| **ABC** | Clasificación ABC |
| **XYZ** | Clasificación XYZ |

## Filtros Disponibles

- **Categoría** - Filtra por categoría de producto
- **Clasificación ABC** - Solo A, B o C
- **Clasificación XYZ** - Solo X, Y o Z
- **Búsqueda** - Por nombre o código

## Detalle de Producto

Al hacer click en un producto verás:

### Información General
- Código y nombre
- Categoría y subcategoría
- Unidad de medida
- Proveedor principal

### Métricas de Venta
- Venta promedio diaria
- Venta últimos 7, 30, 90 días
- Tendencia (crecimiento/decrecimiento)
- Estacionalidad

### Stock por Tienda
Tabla con el stock actual en cada ubicación:
- Tienda
- Stock actual
- Cobertura en días
- Estado (Normal/Bajo/Crítico)

### Historial de Clasificación
Evolución de la clasificación ABC/XYZ en el tiempo.

## Exportación

Exporta el análisis completo a:
- **Excel** - Para análisis adicional
- **PDF** - Para reportes
